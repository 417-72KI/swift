**STRUCT**

# `GitCommit`

> A platform agnostic reference to a git commit.

## Properties
### `sha`

> The SHA for the commit.

### `author`

> Who wrote the commit.

### `committer`

> Who shipped the code.

### `message`

> The message for the commit.

### `parents`

> SHAs for the commit's parents.

### `url`

> The URL for the commit.
