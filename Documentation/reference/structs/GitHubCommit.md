**STRUCT**

# `GitHubCommit`

> A GitHub specific implementation of a git commit.

## Properties
### `sha`

> The SHA for the commit.

### `commit`

> The raw commit metadata.

### `url`

> The URL for the commit on GitHub.

### `author`

> The GitHub user who wrote the code.

### `committer`

> The GitHub user who shipped the code.
